# Interactive Sales Performance Dashboard

## Project Summary
Designed and developed an interactive business intelligence dashboard to help sales managers and executives monitor sales performance, profitability, customer behavior, and regional trends. The project transformed raw transactional sales data into actionable insights through SQL data preparation and Tableau visualizations.

The objective was to provide stakeholders with a centralized dashboard that could answer questions like:

Which regions generate the highest sales?
Which product categories are most profitable?
Which customers contribute the highest revenue?
How are sales and profit changing over time?
Which products need business attention?

The dashboard was built after gathering business requirements and creating dashboard mockups to ensure alignment with stakeholder expectations.

---

## Project Workflow
### 1. Business Requirement Analysis
Started by understanding stakeholder requirements and identifying key business KPIs.

The primary KPIs included:

- Total Sales
- Total Profit
- Total Quantity Sold
- Profit Margin
- Sales Growth
- Customer Performance
- Regional Performance

---
### 2. Data Preparation using SQL
Used SQL to clean and prepare the dataset before importing it into Tableau.

Activities included:
- Removing duplicate records
- Handling missing values
- Formatting date fields
- Creating calculated columns
- Aggregating sales data
- Joining multiple tables
- Filtering unnecessary records
- Preparing summarized datasets

Created dashboard wireframes before implementation to define the layout and user experience.

---
### 3. Dashboard Development in Tableau
Developed multiple interactive dashboards focused on different business perspectives.

Sales Dashboard
Included:

#### KPI Cards
Displayed
- Total Sales
- Total Profit
- Total Quantity
along with Year-over-Year comparison.

#### Sales Trend Analysis
Created line charts showing
- Monthly Sales Trend
- Monthly Profit Trend
to identify seasonal business patterns.

#### Regional Performance
Compared sales across different regions.
Helped identify
- Top-performing regions
- Underperforming regions

#### Category & Sub-category Analysis
Used horizontal bar charts to compare
- Sales
- Profit
across
- Technology
- Furniture
- Office Supplies
and individual product subcategories.

#### Profitability Analysis
Highlighted
- High-profit products
- Loss-making products
using conditional colors.
